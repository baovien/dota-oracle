# Dota Oracle

## Getting Started

### Install

* Clone repository `git clone https://github.com/baovien/dota-oracle` or download `.zip`
* Go to project root `cd dota-oracle`
* Run `pip install .`

### Run

* Start the application by running `python -m doracle.app`
